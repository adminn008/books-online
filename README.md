**AETRIX** é um gerador de tipografia minimalista e estética *low profile* desenvolvido para criadores de conteúdo que buscam personalizar suas biografias e presenças digitais com elegância e mistério.

🔗 **Acesse o projeto:** [https://adminn008.github.io/aetrix-text/](https://adminn008.github.io/aetrix-text/index.html)

---

## 🛠️ Funcionalidades

- **30+ Estilos de Fontes:** De gótico clássico a minimalismo moderno, todos compatíveis com Unicode para Instagram, Discord e X (Twitter).
- **Interface Dark/Luxo:** Design focado em alto contraste e experiência de usuário limpa.
- **Sistema de Contato Inteligente:** Integração em tempo real com **Firebase Realtime Database** para envio de mensagens.
- **Painel Administrativo Privado:** Acesso restrito via senha para gerenciamento de mensagens, controle de leitura e logs de horário.
- **Otimização SEO:** Estruturado para indexação em mecanismos de busca com metatags específicas.
- **Monetização Simples:** Seção de apoio via PIX integrada com sistema de cópia em um clique.

## 🚀 Tecnologias Utilizadas

- **HTML5 & CSS3:** Estruturação semântica e estilização avançada com Google Fonts (*Cinzel* e *Inter*).
- **JavaScript (Vanilla):** Lógica de conversão de caracteres, manipulação de DOM e funções de área de transferência.
- **Firebase:** Banco de dados em tempo real para o sistema de suporte.
- **SVG:** Ícones leves e escaláveis para performance máxima.

## 📁 Estrutura do Repositório

- `index.html`: Core do projeto e gerador de fontes.
- `fale-conosco.html`: Interface de suporte ao usuário.
- `admin.html`: Painel de controle privado para o desenvolvedor.
- `privacidade.html` / `faq.html`: Páginas de suporte e conformidade.
